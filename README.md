# CrunchCo — Snack Ringan Premium

Website landing page modern untuk **CrunchCo**, toko snack ringan premium yang menawarkan berbagai varian rasa autentik Indonesia.

## 🚀 Teknologi yang Digunakan

- **HTML5**: Struktur konten web.
- **CSS3 (Custom Properties & Grid/Flexbox)**: Desain responsif dan modern.
- **JavaScript (ES6+)**: Logika interaktif seperti keranjang belanja.
- **GSAP (GreenSock Animation Platform)**: Animasi performa tinggi untuk pengalaman pengguna yang halus.
- **ScrollTrigger**: Animasi yang dipicu saat pengguna melakukan scroll.
- **Google Fonts**: Tipografi premium menggunakan `Playfair Display` dan `DM Sans`.

## ✨ Fitur Utama

- **Desain Modern & Responsif**: Tampilan yang optimal di berbagai perangkat (Mobile, Tablet, Desktop).
- **Animasi GSAP**: Efek visual yang halus pada hero section, kartu produk, dan elemen scroll.
- **Interaksi Keranjang**: Simulasi penambahan produk ke keranjang dengan notifikasi toast.
- **Struktur Rapi**: Pemisahan file CSS dan JavaScript untuk kemudahan pemeliharaan kode.

## 📁 Struktur Proyek

```text
/
├── css/
│   └── style.css       # File gaya utama
├── js/
│   └── script.js      # Logika interaktif & animasi GSAP
├── snack-store.html   # File HTML utama
├── vercel.json        # Konfigurasi deployment Vercel
└── README.md          # Dokumentasi proyek
```

## 🛠️ Pengembangan Lokal

1. Clone repositori ini.
2. Buka `snack-store.html` langsung di browser Anda atau gunakan extension seperti *Live Server* di VS Code.

## 🌐 Deployment

Proyek ini siap untuk di-deploy ke **Vercel** atau platform hosting statis lainnya.
